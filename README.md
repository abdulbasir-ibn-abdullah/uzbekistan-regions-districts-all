# uzbekistan-regions-districts-all

tumanlar.json faylida O'zbekistonning barcha viloyatlari, jumladan 29.06.2026 holatiga ko'ra barcha tumanlari keltirilgan. Har birlarda (viloyatlar va tumanlarning har birlarida) mos holatda koordinatalar (longitude, latitude) ham keltirib o'tilgan. Ochiq manbalardan foydalangan holatda va maxsus script va AI vositalaridan foydalangan holatda yig'ilgan ma'lumotlarga asoslanib tuzilgan, shu sababli haqiqiy nuqtadan biroz farq qilishi mumkin!
